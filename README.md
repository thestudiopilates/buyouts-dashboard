# Buyouts Dashboard Workspace

This repository contains both the original buyout operations source materials and the new first-party buyout platform scaffold.

## Repository Layout

- `buyout-app/`: source materials, Monday architecture notes, UI prototype, and strategy documents
- `buyout-platform/`: new Next.js and Prisma application scaffold for the first-party buyouts platform

## Recommended Direction

The long-term path is to move buyouts out of Monday and Make into the app in `buyout-platform/`, while keeping `buyout-app/` as migration reference material until cutover is complete.
